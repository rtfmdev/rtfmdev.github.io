    Replase text in template

    Replase TEMPLATE_TEXT with $BUILD_NUMBER var inside filename.json.template and save it into filename.json

    sed "s/TEMPLATE_TEXT/1.$BUILD_NUMBER/g" \
    filename.json.template > filename.json
